評価軸

1. AIエージェントが価値の中心になっているか 単なる機能ではなくAIエージェントが作品の中核として機能しているか、自律的な振る舞い（判断・タスク実行）があるか、“AIエージェントである必然性”があるか等を評価します。
2. 設定した課題へのアプローチ力
設定した課題とその背景、対象となるユーザー、作品が提供する価値といったプロダクトとしてのストーリーに一貫性、妥当性、新規性があるか等を評価します。
3. ユーザビリティ
ユーザーが直観的に使える機能やデザインを有しているかを評価します。
4. 実用性・体験価値の魅力
設定した課題の解決に対する実用性、また作品がどこまで効果的にアプローチできているかを評価します。また提供する体験価値が突き抜けているか等も加点評価します。
5. 実装力
選定した技術や構成の納得度、拡張性や実運用への配慮など、利用を必須としたツールを活用したうえで、これらがどれだけ実現できているかを総合的に評価します。

rule

1. (必須) Google Cloud アプリケーション実行プロダクトの利用
以下のいずれかから1つ以上を選択してください。
   * App Engine・Google Compute Engine
   * Google Kubernetes Engine (GKE)
   * Cloud Run・Cloud Functions (旧 Cloud Functions)
   * Cloud TPU / GPU
2. (必須) Google Cloud AI 技術の利用
以下のいずれかから1つ以上を選択してください。
   * Gemini Enterprise Agent Platform（旧Vertex AI）
      * AIモデルのトレーニング、デプロイ、管理を行うための統合プラットフォーム。AutoML、Vector Search、Explainable AIなどの機能を含む
   * Gemini API
      * 最先端のマルチモーダルAIモデル。Gemini Enterprise Agent Platform（旧Vertex AI）経由での利用を推奨しますが直接利用することも可能
   * Gemma・Imagen・Agent Builder
   * ADK (Agents Development Kit)
   * Speech-to-Text / Text-to-Speech API
   * Vision AI API・Natural Language AI API
   * Translation AI API
3. (任意) その他の技術の利用
   * Flutter
   * Firebase
   * Veo
   * Elasticsearch（スポンサー）
ほか、任意の技術を利用することができます。
