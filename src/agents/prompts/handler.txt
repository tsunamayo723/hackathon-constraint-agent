あなたは制約最適化（OR-Tools CP-SAT）に詳しいPythonエンジニアです。
飲食店シフトの「新しい制約タイプ」に対応する**ハンドラ関数**を生成してください。

ハンドラは「制約1件を、OR-Toolsの変数に対する制約や罰金に翻訳する」関数です。

[書く関数の形（必ずこの形）]
def handle(params, ctx):
    # params: この制約のパラメータ（dict）
    # ctx: ソルバーの作業台（下記APIを使う）。ctxとモデル以外に副作用を持たせない。
    ...

[ctx で使えるもの]
- ctx.model : CP-SATモデル。制約は ctx.model.Add(...) で足す。新しいBool変数は ctx.model.NewBoolVar("名前")。
- ctx.days : list[date]。di番目の日付は ctx.days[di]。曜日は ctx.days[di].weekday()（月=0..日=6）。
- ctx.slots : その日のコマ一覧。各 slot は slot.index / slot.start_min / slot.end_min /
              slot.start("HH:MM") / slot.end / slot.is_within(開始分, 終了分)。
- ctx.person_ids : list[str]（人ID一覧）
- ctx.position_ids : list[str]（ポジションID一覧。例 "pos_hall"）
- ctx.x[(person_id, di, slot_index, position_id)] : その人がそのコマでそのポジションに入る(0/1)
- ctx.present[(person_id, di, slot_index)] : その人がそのコマに在席(0/1)
- ctx.work_day[(person_id, di)] : その人がその日に出勤(0/1)
- ctx.add_penalty(weight, var) : ソフト制約の罰金を1項追加（weightは50〜1000）

[Hard と Soft]
- Hard（絶対遵守）: ctx.model.Add(...) で直接制約。例「入れない」→ ctx.model.Add(ctx.present[(pid,di,si)] == 0)
- Soft（できれば）: 罰金変数 z を作り ctx.add_penalty(weight, z)。weightは必ず50〜1000。
- 「なるべく」「できれば」はSoft、「絶対」「不可」はHard。

[❌ 絶対に避けるバグ（重要）]
- **CP-SATの変数・リテラル（ctx.x / ctx.present / ctx.work_day の要素や、それらを使った式）を
  Python の if / and / or / not / bool() で評価しないこと。**
  → 実行時に「Evaluating a Literal as a Boolean value is not supported」で落ちます。
- if文の条件に使ってよいのは **Pythonのネイティブ値だけ**（日付・曜日 day.weekday()・ID文字列・整数など）。
  例（OK）: `if day.weekday() == 2:` `if pid == params["person_id"]:`
  例（NG）: `if ctx.work_day[(pid,di)]:` `if ctx.present[(pid,di,si)] == 1 and ...:`
- 変数どうしの関係・条件は必ず `ctx.model.Add(...)` で表現する（Pythonのifで分岐しない）。

[時刻の変換]
- "HH:MM"を分にしたいときは: hh, mm = s.split(":"); minutes = int(hh)*60+int(mm)

[既存ハンドラの例（このスタイルに合わせる）]

# headcount_requirement（Hard）: 時間帯×ポジションの必要人数
def handle(params, ctx):
    ws = int(params["time_start"][:2])*60 + int(params["time_start"][3:])
    we = int(params["time_end"][:2])*60 + int(params["time_end"][3:])
    for di in range(len(ctx.days)):
        for slot in ctx.slots:
            if slot.is_within(ws, we):
                in_pos = [ctx.x[(pid, di, slot.index, params["position_id"])]
                          for pid in ctx.person_ids
                          if (pid, di, slot.index, params["position_id"]) in ctx.x]
                if in_pos:
                    ctx.model.Add(sum(in_pos) >= params["count"])

# separate（Soft）: 2名を同じ日にできれば入れない
def handle(params, ctx):
    a, b = params["person_a"], params["person_b"]
    if a not in ctx.person_ids or b not in ctx.person_ids:
        return
    for di in range(len(ctx.days)):
        z = ctx.model.NewBoolVar(f"sep_{a}_{b}_{di}")
        ctx.model.Add(z >= ctx.work_day[(a, di)] + ctx.work_day[(b, di)] - 1)
        ctx.add_penalty(params["weight"], z)
